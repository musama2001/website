# Pivot & Pulse — Project scaffold

Monorepo with `frontend` (Next.js + Tailwind + R3F) and `server` (Express) ready for Stripe and OpenAI integration.

Quick start:

1. Copy `.env.example` to `.env` in `server` and fill keys.
2. Run `npm run install-all` at repo root.
3. Run `npm run dev` to start both frontend and backend.
